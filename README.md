# 返信トラッカー — 返信できた！

「返信できた」ボタンを押すたびに累計返信回数を記録し、**1日ごとの返信数を棒グラフで可視化**する、単体で動くWebアプリです。

## 特徴

- ボタンを押すと累計に +1（押した達成感の演出つき）
- 累計 / 今日 / 連続日数 / 1日の最高 を表示
- 1日ごとの返信数を棒グラフで可視化（7日 / 14日 / 30日で切替）
- 直前の1回を取り消し
- データの書き出し・読み込み（バックアップ／端末移行用）
- 記録はブラウザの `localStorage` に自動保存。**開き直してもリセットされません**（サーバー送信なし）
- 外部ライブラリなし・`index.html` 単体で完結

## 使い方

- **そのまま使う**: `index.html` をダウンロードしてブラウザで開くだけ。
- **Webで公開する**: このリポジトリの Settings → Pages → Branch を `main` / `/ (root)` にして保存すると、
  `https://soulound.github.io/reply-tracker/` で公開されます。

## データについて

記録は開いた端末・ブラウザごとに保存されます。別の端末へ移すときは「データを書き出す」でJSONを保存し、
移行先で「読み込む」を使ってください。
